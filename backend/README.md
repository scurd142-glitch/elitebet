# WRITERSNITE Backend API

Express + TypeScript + PostgreSQL + Prisma + JWT.

## Quick start

```bash
npm install
copy .env.example .env
# Edit .env with DATABASE_URL, JWT_SECRET, CLIENT_URL
npm run db:push
npm run dev
```

API runs at `http://localhost:4000`.
